# 🧠 大五维人格测评 (Big Five / OCEAN)

一个精美的在线大五人格测评工具，纯前端实现，无需后端服务。

## ✨ 特性

- **50 道科学题目** — 五个维度各 10 题，涵盖正反向评分
- **5 点 Likert 量表** — 从「非常不同意」到「非常同意」
- **蜘蛛图 + 柱状图** — Chart.js 驱动的可视化结果
- **详细维度解读** — 高/中/低三个等级的专业描述
- **全中文界面** — 适配中文用户习惯
- **深色主题** — 现代暗色 UI 设计
- **移动端适配** — 响应式布局，手机/电脑均可使用
- **一键分享** — 复制测评结果到剪贴板

## 🚀 快速开始

直接打开 `index.html` 即可使用，或在本地启动一个简单的 HTTP 服务：

```bash
# Python 3
python3 -m http.server 8080

# 或使用 npx
npx serve .
```

然后访问 http://localhost:8080

## 📊 五大维度 (OCEAN)

| 维度 | 英文 | 含义 |
|------|------|------|
| 🎨 开放性 | Openness | 想象力、好奇心、对艺术和新鲜体验的欣赏 |
| 🎯 尽责性 | Conscientiousness | 自律、条理、目标导向与可靠性 |
| 🎤 外向性 | Extraversion | 社交性、活力、积极情绪的表达 |
| 🤝 宜人性 | Agreeableness | 同情心、合作精神、对他人的信任 |
| 🌊 神经质 | Neuroticism | 情绪敏感性、焦虑倾向与心理韧性 |

## 🛠 技术栈

- HTML5 + CSS3 + 原生 JavaScript
- [Chart.js](https://www.chartjs.org/) 数据可视化
- 无需构建工具，零依赖运行
